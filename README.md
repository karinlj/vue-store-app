# Vue STore App

## Description

Vue Store App - Assignment for Ellos.
Built with Vite + Vue.js

![Screenshot](/src/assets/Screenshot.jpg?raw=true "Screenshot")

### Prerequisites

NodeJS

## Installation

To clone

`git clone https://github.com/karinlj/vue-store-app.git`

To install

`cd vue-store-app`

`npm install`

To run

`npm run dev`
